# PulseTabs - Typecho 主题

一款为 Typecho 设计的现代化主题，支持 Tailwind CSS、Font Awesome 6，响应式布局，适合个人博客、内容站。

## 特性
- 基于 Tailwind CSS 3 构建
- 支持深色/浅色模式（可选）
- 响应式设计，移动端适配
- 内置用户中心集成
- 自定义 Logo、导航菜单

## 安装
1. 将 `PulseTabs` 文件夹上传到 `/usr/themes/`
2. 后台启用主题
3. 根据需要修改 `functions.php` 中的配置
4. 和UserCenter 一起使用 https://github.com/24pu/UserCenter
如果要单独使用 去掉 post页面的 收藏 点赞相关代码
和header页面的 用户中心的下来导航
案例展示:https://24pu.com

## 自定义
- Logo 替换：在主题设置中上传
- 配色方案：修改 `tailwind.config.js` 中的 `accent` 色值
